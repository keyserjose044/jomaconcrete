## ✨ What’s New in the 2026 Template

This year’s version is all about **automation, clarity, and visual elegance.**  
Here’s what’s improved:

- 🧠 **Built-in formulas** that handle more logic automatically (no need to calculate sleep hours, nap durations, etc.)
- 🎨 **Unified color formatting system** for better visual flow (each life category now harmonizes)
- ⚙️ **Google Apps Script integrations** for condensed diary logs and automatic daily summaries  
- 🧾 **Dynamic year references** so formulas and graphs automatically shift to 2026  
- 🔍 **Cleaner dropdowns and emoji indicators** for faster input  
- 📅 **Daily image automation ready** (just plug in your Drive folder and script)
- 📊 **Improved Graph Zones** to visualize more metrics (sleep trends, productivity, work hours, happiness, etc.)

Everything was rebuilt from the ground up to make the experience **as intuitive as it is powerful.**

## 🧩 Inside the Template

<figure>
  <img src="/images/rawdatapost/2026-raw-data-sheet.png" alt="2026 Raw Data Template" />
  <figcaption><em>The 2026 edition — polished visuals, optimized structure, and ready for daily use.</em></figcaption>

You’ll find familiar sections — Sleep, Food, Weather, Work, Hobbies, Logs — each refined for usability and automation.

| Category | Columns | Notes |
|-----------|----------|-------|
| 💤 **Sleep** | D–G | Auto-calculates hours, nap tracking, color-coded for late nights |
| 🍽 **Meals** | I–K | Dropdowns + symbols for meal type and quality |
| 🌤 **Weather** | N–O | Highs, lows, color-coded by extremes |
| 📔 **Diary Logs** | Q–R | Manual + AI-generated summaries |
| 💼 **Work** | T–U | Task description, hours worked, category filters |
| 🎧 **Audiobooks** | AD–AF | Track listening time and new titles |
| 🧘‍♂️ **Habits** | W–Z | Religious study, languages, exercise, etc. |

Conditional formatting now uses **a unified symbol palette**:  
`@` = Highlight, `"` = Milestone, `$` = Achievement, `#` = Warning.  
Each instantly changes the cell color for better visualization.
